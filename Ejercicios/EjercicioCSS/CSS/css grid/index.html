Sobre mí

Les doy la bienvenida a mi sitio, un espacio de intercambio y desarrollo. En el busco compartir mi avance en el mundo del desarrollo web y como solo las ganas son suficientes para iniciarse en cualquier área. Curiosamente, el interes en este tópico, surgió a medida que afrontaba diversos desafíos en el mundo laboral. No fue hasta hace algunos años que me ví orillada a conocer más acerca de los fundamentos del WordPress, ya que debía actualizar información para el staff a traves de una página en este formato. Fue todo un reto, pues poco conocía respecto a la programación. Este fue mi punto de partida para acercarme al universo de la programación. Quería conocerlo todo, pero esto significaba un desafio sin precedentes. Desde aquel momento, busque avanzar y profesionalizarme en este aspecto. Hoy, a mis 39 años, y con muchas ganas de desarrollarme en el ambiente, inicie la carrera de Desarrollo Web y aplicaciones digitales, dictada de forma virtual en el Instituto Santa Rosa de Calamuchita. Actualmente, curso el primer año y me encuentro muy feliz de poder compartir con ustedes el avance que he tenido en tan poco tiempo.

Mis proyectos

En esta sección podrán encontrar algunos de los trabajos que he realizado en compañía de otros aspirantes y desarrolladores a quienes le agradezco la colaboración

Proyecto 1

Mi primer proyecto en el mundo del desarrollo web fue una página creada en HTML. En este proyecto, me enfoqué en mostrar los fundamentos principales del lenguaje HTML, incluyendo la estructura básica de una página web, el uso de etiquetas para definir encabezados, párrafos, listas y enlaces, así como la incorporación de imágenes y tablas. Este proyecto me permitió comprender cómo HTML sirve como el esqueleto de cualquier sitio web, proporcionando la estructura necesaria para que el contenido sea presentado de manera organizada y accesible.

Proyecto 2

En mi segundo proyecto, me dediqué a aprender y practicar la creación de funciones en JavaScript. Este proyecto se centró en los conceptos básicos del lenguaje, como la definición y ejecución de funciones, el manejo de variables y la implementación de lógica condicional. Aunque no vinculé estas funciones a un sitio web, este ejercicio me permitió comprender cómo JavaScript puede ser utilizado para realizar cálculos, manipular datos y automatizar tareas, sentando una base sólida para futuros proyectos más complejos.

Proyecto 3

En mi tercer proyecto, me adentré en el mundo del CSS para darle estilo a una página web. Este proyecto se centró en los fundamentos del lenguaje CSS, como la aplicación de estilos a través de selectores, la manipulación de colores, fuentes y márgenes, y la creación de un diseño visualmente atractivo y responsivo. Aprendí a utilizar las propiedades de CSS para mejorar la presentación de los elementos HTML, logrando una página más estética y funcional.
